# Navigating

Navigating on RetroArch is as easy as you can imagine. Depending on the device
you have, the control scheme is determined automatically. For example, if you
are going to use it on a computer, the basic key combinations of the keyboard
will apply. You move with
<kbd>↑ Up</kbd><kbd>↓ Down</kbd><kbd>← Left</kbd><kbd>→ Right</kbd> and select
with <kbd>⏎ Enter</kbd>, and you return with <kbd>⌫ Backspace</kbd>. If you plug
a controller, you can also move with the D-Pad buttons.

The default [GUI][gui] for RetroArch is [Ozone][ozone]. While the "skin" may
differ on certain platforms, basic layout and navigation remains broadly the
same.

## PC

By default the keyboard should be ready to use.

Using <kbd>← Left</kbd> and <kbd>→ Right</kbd>, you move to a different tab,
each tab will relate to an aspect of the RetroArch experience (Main menu for
game loading, Settings for configuration, etc.). Once the desired tab is
selected, use <kbd>↑ Up</kbd><kbd>↓ Down</kbd> to move to an option.

To advance in the menu press <kbd>⏎ Enter</kbd> (or <kbd>⏎ Return</kbd>). To go
back to the previous menu use <kbd>⌫ Backspace</kbd>.

Whilst in a game, use <kbd>F1</kbd> to open the [Quick Menu][quick-menu].

Pushing <kbd>⎋ Esc</kbd> twice will quit the program.

### Main Menu

![Ozone Main Menu][ozone-main-menu]

Welcome to the main menu. This is where you will launch games, download and
update cores, and have access to freely available content. **`Load Core`**,
**`Load Content`** and the **`Online Updater`** are the menu items you will
probably be using the most.

!!! abstract "See also"
    [**Starting a Game**][starting-a-game]

### Settings

![Settings][ozone-settings]

Here you will find all the options available to configure RetroArch, from
Graphics, Input, Sound and everything else.

### Favorites

![Favorites][ozone-favorites]

Favorites allow yous to maintain a short list of games for quick access. You can
add content to this playlist from the [Quick Menu][quick-menu] or before
launching it.

### History

![History][ozone-history]

Recently launched content will be added to this Playlist; the content you ran
last will be at the top.

### Images, Music and Videos

These tabs will handle all your media. Images will contain any screenshots
you've made.

### Netplay

RetroArch relies on peer-to-peer networking to reduce network latency and ensure
the best possible experience. It allows multiplayer over the internet. The user
can host or join a network gaming session, or use the Spectator mode to watch
others play. RetroArch has a built-in netplay lobby.

### Import content

![import-content][ozone-import]

Either you scan automatically, so that any content that matches the internal
database will be added to Playlists, or you scan manually, not being dependent
on the database so you can freely add your content to Playlists.

## Mobile

The mobile interface, [GLUI][glui], is more minimalist compared to the PC's
interface. Touch is the default input, although depending on device other
options may be available. There are three different tabs, which can be switched
at the bottom of the menu.

### Main Menu

![GLUI Main Menu][glui-main-menu]

The main menu will be where you launch games, download and update cores, start
or join netplay sessions and pretty much anything to do with gaming.

!!! abstract "See also"
    [**Starting a Game**][starting-a-game]

### Playlists

Handles anything to do with creating and maintaining playlists. It gives you
scanning options to recognize your game collection. It also provides access to
your images, music and videos.

### Settings

Here you will find all the options available to configure RetroArch, from
Graphics, Input, Sound and everything else.
